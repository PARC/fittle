
#### agents.json

This file tells the Fittle app how to communicate with a Coach agent (via a Web Universal Resource Locator (URL).

It is specified by a Coach Designer:

```
{
    "name": "NameForAgent",
    "url": "http://example.com",
    "events": [
      "start",
      "stop",
      "report"
    ],
    "active": true
}
```
