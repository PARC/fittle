How system can be deployed on Heroku:
Got to heroku.com and create an app and add the mLab mongo database
From local project git repository:
heroic login
heroku git:remote -a parc-positive-day
heroku config:set MONGO_URL=mongodb://parcuser:xx@HOSTNAME.mlab.com:21010/heroku_mqft0h64
heroku config:set ROOT_URL=https://HOSTNAME.herokuapp.com
heroku buildpacks:set https://github.com/AdmitHub/meteor-buildpack-horse.git
git push heroku master
